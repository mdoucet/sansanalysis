About SANS Data Analysis

Mission statement

The mission that has driven the development of this site can be described as the following:

1- Create scentific software using current software development technology:

Software applications are continuously moving from deskptop applications to web applications. We believe that the same is true for scientific software. Proper software engineering practices and modern tools will help build the applications that will push science forward.

2- Provide tools for scientific collaboration:

Science is a collaborative process. Now that the technology is readily available, we believe that scientists should have analysis tools that also support the level of collaborative efforts needed for their work.

What is SANS?

If you don't know what small angle scattering is, chances are you will not enjoy this site. You're not missing much: it's all pretty boring really.


What is the path forward for this site?

In terms of data analysis, the current version of this SANS analysis web site only provides the computation of P(r) inversion. While doing so it provides users with data management and collaboration tools. This is a first step. Following your feedback, data modeling and fitting will soon be available. Most importantly, we want to hear from you, the users, to tell us what functionality you would like to use.
